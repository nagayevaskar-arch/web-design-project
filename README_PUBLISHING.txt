HOW TO PUBLISH THIS WEBSITE WITH GITHUB PAGES

1. Open github.com and create a new repository.
2. Upload all files and folders from this project folder.
3. Make sure index.html is in the main/root folder of the repository.
4. Open Settings > Pages.
5. In Build and deployment, choose Deploy from a branch.
6. Choose branch: main and folder: /root.
7. Press Save.
8. Wait 1-3 minutes. GitHub will give you a public website link.
9. Open the link and check all pages: Home, Destinations, Culture, Contact.

IMPORTANT:
The contact form uses FormSubmit. After the first form submission, FormSubmit may send an activation email. Confirm it once, then messages will be delivered normally.
